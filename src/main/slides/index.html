The jQuery-ish Navigator API

  • CSS Selectors 
    
$("div.vectorTabs a[accesskey='e']")
  • Indexes 
    
$("h2", 1)
  • Attribute and Text Matching 
    
$('input', placeholder: 'Search')
$('a', text: 'Read')
$('a', title: contains('edit this page'))
  • Finding & Filtering 
    
$('#toc').find('li', 1)
$('li').filter('.toclevel-1')
$('#toc span').not('.tocnumber')
$('h2').has('.mw-headline')
